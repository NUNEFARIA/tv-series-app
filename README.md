# TV Series Manager

Aplicação desktop desenvolvida em Java Swing para gerenciamento de séries de TV.

## Funcionalidades

- Buscar séries pela API TVMaze
- Adicionar aos favoritos
- Marcar como assistida
- Adicionar à lista Quero Assistir
- Remover séries das listas
- Visualizar detalhes
- Ordenar listas
- Persistência em JSON
- Alteração de nickname

## Tecnologias

- Java 17
- Swing
- Gson
- TVMaze API

## Arquitetura

- View
- Controller
- Service
- Repository
- Model
